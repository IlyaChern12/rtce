# Real-Time Collaborative Editor

**Описание:**  
RTCE — это проект редактора документов с поддержкой многопользовательского редактирования в реальном времени. На данный момент реализован backend с регистрацией и аутентификацией пользователей, CRUD для документов и базовая заготовка WebSocket-сервера.

## Текущий функционал

- Регистрация (`POST /register`) и авторизация (`POST /login`) с JWT.  
- Эндпоинт `/me` для проверки токена.  
- CRUD документов:
  - `POST /documents` — создание документа.  
  - `GET /documents` — получение списка документов пользователя.  
- WebSocket `/ws` — минимальный эхо-сервер для тестирования real-time соединений.

## Стек технологий

- **Backend:** Go, REST API, JSON, PostgreSQL, Redis, WebSocket  
- **DevOps:** Docker, docker-compose, Makefile  
- **Дополнительно:** JWT авторизация, middleware для токена
